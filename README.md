# 화성시 산불 위험 순찰 지도

화성시 내 산불 위험 지역을 지도에 표시하고, 차량으로 순찰 가능한 추천 노선을 제공하는 웹 서비스입니다.

## 주요 기능

1. 일반 지도에서 잘 보이지 않는 산 위치 표시
2. 산별 위험 점수와 위험 순위 확인
3. 농경지, 공장, 등산로, 과거 산불 이력 등 위험 요소 시각화
4. 차도 기반 순찰 추천 노선 5개 제공
5. 선택한 노선의 이동 거리와 커버 산 정보 확인

## 작동 방식

1. 화성시 산 위치 데이터를 정리합니다.
2. 산 주변의 농경지, 공장, 등산로, 과거 산불 이력 데이터를 결합합니다.
3. 위험 요소를 기준으로 산별 위험 점수를 계산합니다.
4. 도로 노드와 링크 데이터를 활용해 효율적인 차량 순찰 노선을 생성합니다.
5. 사용자가 지도에서 산을 클릭하면 해당 산의 위험 점수, 위험 순위, 주변 위험 요소 정보를 확인할 수 있다.
## 사용 기술

프론트엔드: HTML, CSS, JavaScript, Leaflet

백엔드: 없음

데이터 처리: Python, pandas, geopandas, shapely, pyproj

지도 데이터: OpenStreetMap, 공공 데이터 기반 산림·도로·농경지·공장·산불 이력 데이터

노선 추천: Python 직접 구현, Dijkstra 최단경로 알고리즘, Greedy 위험도 커버 방식

## 실행 방법

`index.html` 파일을 브라우저에서 열어 실행합니다.

메인 지도 원본은 `hwaseong_fire_patrol_map.html`에서 확인할 수 있습니다.
